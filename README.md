# cwdc
CWDC - Change Working Directory and execute the command
<hr>
<h2>Install:</h2>
<b>git</b> clone https://github.com/Pr00x/cwdc.git<br>
<b>sudo make</b> clean install
<hr>
<h2>Usage:</h2>
<b>cwdc</b> -d <b>[</b>PATH<b>]</b> -c <b>[</b>COMMAND<b>]</b>
